# 🏓 Jogo de Ping Pong - Lógica de Programação

Este projeto é um clássico jogo de Ping Pong desenvolvido durante os meus estudos de lógica de programação na plataforma **Alura**. O objetivo principal foi aplicar conceitos fundamentais de desenvolvimento de software em um ambiente visual e interativo.

---

## 🚀 Tecnologias Utilizadas

- **JavaScript:** Linguagem principal para a lógica do jogo.
- **p5.js:** Biblioteca focada em criação visual e animações.
- **HTML5/CSS3:** Estrutura e estilização básica do ambiente de execução.

---

## 🧠 Conceitos Aplicados

Durante o desenvolvimento, explorei pilares essenciais da programação:

- **Variáveis e Operadores:** Controle de posição, velocidade e pontuação.
- **Condicionais (If/Else):** Implementação da física do jogo (quicar nas bordas e detecção de colisões).
- **Funções:** Organização do código em blocos reutilizáveis (`drawBall`, `movePaddle`, etc).
- **Loops e Frames:** Entendimento de como o jogo é renderizado continuamente na tela.
- **IA Simples:** Lógica para a raquete do oponente seguir a posição da bolinha automaticamente.

---

## 🎮 Como Jogar

1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Utilize as setas do teclado (**Cima** e **Baixo**) para controlar sua raquete.
4. O objetivo é marcar pontos fazendo a bolinha passar pela raquete do oponente.

---

## 🛠️ Estrutura do Projeto

- `index.html`: Estrutura base que carrega a biblioteca p5.js.
- `sketch.js`: Onde reside toda a lógica de movimento, colisão e pontuação.
- `style.css`: Estilização básica do canvas.
